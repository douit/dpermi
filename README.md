# Dpermi

#### 项目介绍
Dpermi为采用SpringBoot+Mybatis+Redis+Bootstrap+Vue.js搭建的数据权限系统，可实现使用数据表任一字段来进行权限控制，同时还有方法权限控制和权限有效期控制功能。

#### 功能介绍
##### 菜单：搜索、添加、修改功能；可以自由定制无限级菜单。

![输入图片说明](https://images.gitee.com/uploads/images/2018/0905/160931_7c0708aa_2041403.png "07.png")

##### 用户：搜索、添加、编辑、角色功能；可以查看、添加、删除用户的角色。
![输入图片说明](https://images.gitee.com/uploads/images/2018/0905/162134_b46b5d02_2041403.png "08.png")
![输入图片说明](https://images.gitee.com/uploads/images/2018/0905/162148_73123916_2041403.png "06.png")

##### 角色：搜索、添加、修改、菜单权限、数据权限、用户功能；菜单权限是为角色授予菜单权限及页面功能权限，数据权限是自定义的数据表权限（又称数据行权限），用户功能是为指定角色查看、添加和删除用户。
